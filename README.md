# ☕ Peak Coffee - Application de Commande de Café

Peak Coffee est une interface web moderne et responsive permettant de personnaliser et commander son café en quelques clics. Ce projet a été réalisé dans le cadre de ma formation **BTS SIO SLAM**.

## 🚀 Fonctionnalités
- **Personnalisation complète :** Choix de la saveur (Pistache, Noisette, Rhubarbe), de la taille et des extras.
- **Panier dynamique :** Ajout d'articles en temps réel avec calcul automatique du prix total.
- **Expérience Utilisateur (UX) :** Interface fluide avec système de "flashcards" et retour visuel immédiat.
- **Validation de commande :** Fenêtre modale de résumé avant le paiement fictif.
- **Responsive Design :** Entièrement optimisé pour une utilisation sur smartphone et tablette.

## 🛠️ Technologies utilisées
- **HTML5** : Structure sémantique.
- **CSS3** : Flexbox, animations de transition et design moderne (Coffee Style).
- **JavaScript (Vanilla)** : Manipulation du DOM, gestion du panier et logique de calcul.



## 🔮 Évolutions à venir (Roadmap)

Le projet Peak Coffee est une première étape "Front-End". Une **version Full Stack** est actuellement en cours de réflexion pour transformer cette interface en un véritable outil de gestion commerciale :

- **Base de données relationnelle (SQL) :** Intégration d'une base de données pour stocker le catalogue de produits (cafés, saveurs, prix) et ne plus dépendre uniquement du code HTML.
- **Gestion des Commandes :** Persistance des données pour enregistrer chaque commande client de manière sécurisée.
- **Tableau de bord Administrateur :** Développement d'une interface permettant au gérant de modifier les stocks et les tarifs en temps réel via la base de données.
- **Système d'Authentification :** Possibilité pour les clients de créer un compte pour retrouver leurs préférences.
